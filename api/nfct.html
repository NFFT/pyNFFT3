class pyNFFT3.NFFT(N, M, n=None, m=8, f1=None, f2=c_uint(65))¶

Bases: object

A class to perform non-equispaced fast Fourier transforms (NFFT) considering a D-dimensional trigonometric polynomial. Just N and M are required for initializing a plan.
