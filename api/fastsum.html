pyNFFT3.fastsum - fastsum Class¶

fastsum Methods¶

class pyNFFT3.FASTSUM(d, N, M, kernel, c, n=256, p=8, eps_I=0.03125, eps_B=0.0625, nn=512, m=8)¶

Class to perform the fast summation algorithm:

\[f(\pmb{y}) \coloneqq \sum_{k = 1}^{N} lpha_k \, \mathscr{K}(\pmb{y} - \pmb{x}_k) = \sum_{k = 1}^{N} lpha_k \, K (\lVert \pmb{y} - \pmb{x}_k \]

Vert_2)

for given arbitrary source knots \(\pmb{x}_k \in \mathbb{R}^d\), where \(k = 1, 2, \ldots, N\), and a given kernel function :math:`mathscr{K}(cdot) = K (lVert cdot

Vert_2)`, where \(\pmb{x} \in \mathbb{R}^d\), which is an even, real univariate function that is infinitely differentiable at least in \(\mathbb{R} \setminus \{ 0 \}\).

If \(K\) is infinitely differentiable at zero as well, then \(\mathscr{K}\) is defined on \(\mathbb{R}^d\) and is called a nonsingular kernel function. The evaluation is performed at \(M\) different points \(\pmb{y}_j \in \mathbb{R}^d\), where \(j = 1, \ldots, M\).

Just d, N, M, kernel, and c are required for initializing a plan.
